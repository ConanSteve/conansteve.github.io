通过zip命令压缩和解压缩文件

Created2022-04-15|Updated2022-04-18|LinuxLinux命令
|Word count:495|Reading time:1min|Post View:

通常 zip 已经安装，但验证下也没坏处。你可以运行以下命令来安装 zip 和 unzip。如果它尚未安装，它将立即安装。

zip

  • -d : 从 压缩文件内删除指定的文件

  • -D : 压 缩文件内不建立目录名称

  • -q : 不显 示指令执行过程

  • -r : 递 归处理，将指定目录下的所有文件和子目录一并处理

  • -S : 包 含系统和隐藏文件

  • -<压缩效率> 压 缩效率是一个介于1-9的 数值

  • -b : 创建zip文件临时目录

分卷压缩和合并解压

unzip

  • -j : 目录结构不重新创建
  • file[.zip]:待解压的包
  • [file(s) …]：要解压的文件，默认解压全部文件
  • -d : 解压的输出目录,默认当前目录

zipsplit

命令用于将较大的“zip”压缩包分割成各个较小的“zip”压缩包。

**语法格式：**zipsplit [参数]

常用参数：

-n 指定分割后每个zip文件的大小
-t 报告将要产生的较小的zip文件的大小
-b 指定分割后的zip文件的存放位置
